cf-371-c

发表于 2017-07-03

  • 题目链接: http://codeforces.com/contest/371/problem/C

  • 题目大意: 给你一个字符串表示一个汉堡的组成，你初始有一些材料，然后给你其他材料的价格和你初始有多少钱，最后求最多能做出多少汉堡来

  • 思路: 需要花多少钱的函数F(x) = max(0ll,kB*x - kNb)*kPb + max(0ll, kS*x-kNs)*kPs + max(0ll, kC*x - kNc)*kPc 这个函数是一个单调函数，二分即可求出答案，二分注意区间开闭[left, mid), [mid, right).

cf-814-c An impassioned circulation of affection

发表于 2017-06-09

  • 题目链接: http://codeforces.com/problemset/problem/814/C

  • 题目大意: 给你一个原始字符串，有Q次询问， 每次询问可以涂m次ch颜色，问每次涂颜色后m颜色的最大连续长度，每次都是在原始串上操作

  • 思路: 尺取法,left表示在当前位置能得到的最大长度,如果kStr[left] == ch 那么当前以 left + 1开始的最大长度一定 < 以left开始的最大长度.如果kStr[left] != ch, 那么当前以left + 1开始的最大长度 一定 >= 以left开始的最大长度.
